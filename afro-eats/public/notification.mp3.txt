# Notification Sound Placeholder

This file indicates where to place your notification sound file.

**Instructions:**
1. Find or create a notification sound file (MP3 or WAV format)
2. Rename it to: notification.mp3
3. Place it in: afro-eats/public/notification.mp3
4. Delete this .txt file

**Suggested sources for notification sounds:**
- https://mixkit.co/free-sound-effects/notification/
- https://freesound.org/ (search "notification" or "ding")
- Record your own using your phone or computer

**Recommended specifications:**
- Format: MP3 or WAV
- Duration: 1-3 seconds
- Volume: Medium (not too loud)
- Type: Pleasant "ding" or "chime" sound
